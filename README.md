# Transpose-2D-Matrix
Python script to Transpose a 2D matrix.

Input any 2D array or a matrix and script will output a transposed matrix.
Wriiten on python3

Eg:Input :  lst = [[6, 5, 1, 0],
                  [3, 1, 0, 2],
                  [5, 6, 4, 7]]

Output : lst = [[6, 3, 5],
               [5, 1, 6],
               [1, 0, 4],
               [0, 2, 7]]
